# Mapa-com-RPPNs
Mapa interativo montado para a matéria que escrevi no Medium: https://shorturl.at/fsQV8

A muitos anos atrás me deparei com um gráfico com o número de RPPNs (Reserva Particular do Patrimônio Natural – Um tipo específico de Unidade de Conservação, uma área protegida por lei).
Desde então uso este gráfico em aulas para expressar como NÃO fazer um gráfico.

Neste post sugiro alternativas ao gráfico de pizza e apresento os códigos para montar um mapa do Brasil interativo com as frequências por estado
